Detailed Description

Base model controller.

Implements common functionality and attributes which otherwise would be repeated through all the classes. The interesting bits are creation of items with initWithAPIDictionary: (FLContent_item) and get_controller: (FLContent_item), which returns the most appropriate view controller to show the specific item.
