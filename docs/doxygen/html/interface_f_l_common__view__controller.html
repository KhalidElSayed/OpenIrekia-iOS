- (void) show_error_in_more_cell: (NSError*)  error
more_item: (FLMore_item*)  more_item 

Updates the "more cell" with a network error.

In order to update the more cell, you have to pass the underlaying logical FLMore_item which stores its state. Pass it as the more_item parameter. Note that you can pass anything there (even nil), the method will simply do nothing if an FLMore_item cell is not provided.
