Detailed Description

Handles the custom content of a news cell.

News cells usually have a title, a text, a thumbnail image and a footer.

Uses custom drawing to show the image and text. Based on code found at http://blog.atebits.com/2008/12/fast-scrolling-in-tweetie-with-uitableview/.
