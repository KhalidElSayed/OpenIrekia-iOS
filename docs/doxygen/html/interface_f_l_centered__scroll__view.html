Detailed Description

Tricky hack to have images center in the middle.

Comes from https://devforums.apple.com/message/9501#9501 At some point this should be ditched and replaced by other open source photo viewers available which don't have a zooming in/out stutter. Or review that thread to see if there are new solutions.
