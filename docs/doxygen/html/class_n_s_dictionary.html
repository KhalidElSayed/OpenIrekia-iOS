- (NSString *) get_url: (NSString*)  key
def: (NSString*)  def 
[implementation]

Wrapper over get_string which tries to clean up spaces and stuff.

Note that this does NOT a full url encoding, because doing so we would then translate also the path separators, colons, etc. This is pretty much a "try to parse a few bad characters without breaking the rest", since url encoding is not "reentrant".

Returns:
Returns the value of the string, as much clean and URL valid as possible.
