Detailed Description

Handles a facade view for a video.

Unlike the FLGallery_view_controller or the FLNews_view_controller which handle groups of items, the FLMovie_view_controller handles a single item. The reason is that videos cannot me "manipulated". You need a previous facade screen to allow the user see the description or share the video without actually having to download it.

The view will show some preview image and text before loading the movie.
