Detailed Description

Special wrapper class to avoid strange frame positioning.

Just like in the FLCentered_scroll_view setFrame method, this class also suffers from 'magical' frame resetting calls. Just after the controller is being pushed on the navigation the frame is reset to the screen size, which is good for many cases/controllers, but not ours.

So we override the normal view with this class to force a specific rectangle positioning, specified by the FLPhoto_view_controller::resize_frame method.
