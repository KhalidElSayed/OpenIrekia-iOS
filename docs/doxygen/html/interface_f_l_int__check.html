Detailed Description

Class to mark integers as used.

You can create a class yourself or get a special domain class with the cache() method. The purpose of the class is to have a memory persistent store of integers that have been used. This list is useful for the FLItem_view_controller. That class needs to refresh cached data from the network, even if it was loaded from disk. But only once per application run.

Since network news elements are replaced every time the user asks for a network refresh, the FLNews_item class can't hold the state of having been downloaded before. So we use this class to store that info.

Ugly, but works. As long as I don't forget what this was meant for...
